## 项目简介

本项目基于 EcoPaste 进行二次开发，致力于打造一款跨平台但 Windows 优先的基于工作流的桌面效率工具。

除了继承并增强 EcoPaste 原有的强大剪贴板管理功能外，CueFlow 还计划整合截图贴图、待办事项、屏幕录制等多样化功能，全方位提升用户的桌面工作效率。

## 功能介绍

- 🎉 基于 Tauri v2 开发，轻量高效，跨平台体验更进一步。
- 💻 支持 Windows、macOS 和 Linux（x11），在多设备间无缝切换。
- ✨ 简洁直观的用户界面，操作简单，零门槛上手，开箱即用。
- 📋 支持纯文本、富文本、HTML、图片和文件类型的剪贴板内容。
- 🔒 数据本地存储，确保用户隐私安全，数据完全掌控在用户手中。
- 📝 支持备注功能，轻松分类、管理和检索，让工作更高效。
- ⚙️ 丰富的个性化设置，满足不同用户需求，打造专属体验。
- 🤝 完善的文档与社区支持，与开发者共同探索与成长。
- 🧩 持续优化中，更多惊喜功能等你发现。
- 🎨 界面将逐步迁移至Material Design，带来更现代化的视觉体验。

## 愿望清单

以下是计划在未来版本中实现的功能和改进：

- [ ] 📸 截图贴图功能，快速捕获并分享屏幕内容
  - [ ] 区域截图与全屏截图
  - [ ] 截图编辑（涂鸦、标注、马赛克）
  - [ ] 贴图功能（固定在桌面）
  - [ ] 快捷键支持

- [ ] ✅ Todo功能，集成待办事项管理，提高工作效率
  - [ ] 任务分类与标签系统
  - [ ] 任务优先级与截止日期
  - [ ] 任务提醒与通知
  - [ ] 与日历应用集成

- [ ] 🎬 录屏功能，一键录制并分享操作过程
  - [ ] 区域录制与全屏录制
  - [ ] GIF与MP4格式支持
  - [ ] 录音选项
  - [ ] 简单编辑与裁剪

- [ ] 🚀 启动器功能，快速访问常用应用和功能
  - [ ] 自定义启动项分类
  - [ ] 快捷键绑定
  - [ ] 搜索功能
  - [ ] 最近使用记录

- [ ] ☁️ 云同步功能，跨设备无缝同步剪贴板内容
  - [ ] 多云平台支持（OneDrive、Google Drive、iCloud）
  - [ ] 自动同步与手动同步选项
  - [ ] 同步历史与冲突解决
  - [ ] 端到端加密

- [ ] 📱 移动端支持，在手机和平板上使用CueFlow
  - [ ] iOS应用
  - [ ] Android应用
  - [ ] 跨设备剪贴板同步
  - [ ] 移动端专属UI设计

我们将根据社区反馈持续调整和优化这些计划。如果您有其他功能建议，欢迎通过Issue提出！
